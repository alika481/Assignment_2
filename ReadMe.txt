() - Event Binding is used to pass the data from html to ts.
[] - Property Binding syntax is used to pass the data from ts to html
[()]- This is called as banana inbox syntax which is used for 2 way data binding.
      But in order to use 2 way data binding , we have to use a built in module 
      known as FromsModule.
Two way data binding is very important feature of angular and very popular.
All form submission is typically done using two way data bindig syntax.
[(ngModel)]="username" 
where ngModel is a built in attribute given by angular,
      username is a field defined in the ts file
